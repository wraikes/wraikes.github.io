Guns: A Look into Homicides, Suicides and Accidental Deaths

Data Loading

The data is fairly clean, so not much processing is needed. A single record with an NA value for intent was omitted, solely because it wont help with the analysis.

library(zoo)
library(lubridate)
library(tidyverse)
library(nnet)
library(pscl)

setwd('~/Programming_Projects/Guns')
guns <- read_csv('guns.csv')[, -1]

guns <- filter(guns, intent != 'NA')
guns$date <- as.yearmon(make_date(year = guns$year,  month = guns$month))

Overall Yearly Trends by Intent

First, we take a look at the overall yearly trends for deaths by intent. The trends look fairly flat over the three years (despite the apparent seasonality). Homicide deaths hover around 1,000 per month, while suicides are around 1,750.

guns %>%
  group_by(date, intent) %>%
  summarise(n = n()) %>%
  ggplot() +
    geom_line(mapping = aes(x = date, y = n, color = intent)) +
  labs(title = 'Overall Yearly Trends by Intent', x = 'Year', y = 'Count')

It should be noted that the data is recorded by month. So the seasonality may just be due to the fact that different months are composed by a different number of days. For instance, February only has 28 days in a month (except for Leap years).

Overall Monthly Trends by Intent

The plot below rectifies this to some degree. The plot depicts the average number of deaths by intent per day for any given month. This means that the different number of days in February vs.December, for instance, will not have a factor into the supposed seasonality witnessed in the first plot.

As we can see, February is indeed a month that has less homicides than other months; and suicides are less likely to happen during the month of December. On the other hand, the summer months have registered a larger portion of homicides and suicides.

guns %>%
  group_by(date, intent) %>%
  filter(intent == 'Homicide' | intent == 'Suicide') %>%
  summarise(n = n()) %>%
  mutate(month = factor(month(date)), 
         days = days_in_month(date)) %>%
  group_by(month, intent) %>%
  summarise(total = sum(n), days = sum(days)) %>%
  mutate(days_avg = total/days) %>%
  ggplot() + 
  geom_col(aes(x = intent, y = days_avg, fill = month), color = 'white', 
           position = 'dodge') + 
  labs(title = 'Avg Deaths Per Day for Each Month', x = 'Intent', 
       y = 'Count')

Intent Based on Demographics

Next, a multinomial model is fit based on the three different intents: homicide, suicide and accidental - the latter is the reference category. The intent undetermined was omitted because it did not add to the analysis.

The analysis behind the multinomial model is to determine which variables may have a relationship with the different types of intent.

guns_multi <- guns %>%
  filter(intent != 'Undetermined') %>%
  mutate(place = ifelse(place == 'Home', 'Home', 
                        ifelse(place == 'Street', 'Street', 'Other')),
         education = factor(education),
         race = relevel(factor(race), ref = 'White')) %>%
  dplyr::select(-hispanic, -date, -year, -month)

guns_multinom <- multinom(intent ~ ., data = guns_multi)

The coefficients are detailed below. At first glance, its noticeable that police involvement is only an area that impacts homicides. Otherwise, its a little hard to tell the relationship between the independent and dependent variables.

Instead of trying to decipher the coefficients, well plot the ratios and their confidence levels to get a better idea of the relationships. But first, I want to note that its a little tricky to understand how to interpret the model.

For instance, if we skip ahead to the error bar chart, we will notice that males are less likely to be a victim of homicide compared to suicide or accidental deaths, when compared to the same proportions for females. This is not to say that males are less likely to be a victim of homicide compared to females.

The chart below details this nuance. More than 85k males died from a firearm, while only 14k females suffered similar fates. Yet only 34.7% of the deaths for males were homicides, and likewise, 37.6% of the deaths for females were homicides. Although more men died from homicides overall, proportionately, they were less likely to die from a homicide compared to females.

When following this same logic, we can decipher more trends among different demographics. First, we can see that homicides inflict minorities more often than White individuals (the reference variable). The street is unsurprisingly a common place for homicides to take place (the reference variable is at home). The more educated a person is, the more likely suicides occurs. And lastly, age does not appear to be a factor in homicides, suicides or accidental deaths.

errors <- tibble(vars = rep('NA', 14),
                 upper_homicide = rep(0, 14), 
                 lower_homicide = rep(0, 14),
                 upper_suicide = rep(0, 14),
                 lower_suicide = rep(0, 14))

errors$vars = colnames(coef(guns_multinom))
errors$upper_homicide = exp(coef(guns_multinom) + 1.96 * summary_multinom$standard.errors)[1, ]
errors$lower_homicide = exp(coef(guns_multinom) - 1.96 * summary_multinom$standard.errors)[1, ]
errors$upper_suicide = exp(coef(guns_multinom) + 1.96 * summary_multinom$standard.errors)[2, ]
errors$lower_suicide = exp(coef(guns_multinom) - 1.96 * summary_multinom$standard.errors)[2, ]

error <- errors %>%
  gather(intent, value, -vars) %>%
  separate(intent, c('level', 'intent')) %>%
  spread(level, value) %>%
  filter(vars != 'police', vars != '(Intercept)')

ggplot(data = error) +
  geom_errorbar(aes(x = vars, ymin = lower, ymax = upper, color = intent)) + 
  coord_flip() +
  scale_y_continuous(breaks = seq(0, 11, 1)) + 
  geom_hline(yintercept = 1) + 
  labs(title = 'Confidence Intervals for Multinomial Coefficients', x = 'Variables', 
       y = 'Coefficients')

Model Fit

Lastly, we do a quick check to see how well the model fits. Its an ungrouped multinomial model, so most diagnostics dont apply. However, below is the McFaddens pseudo-R2 value of 0.39 - heuristically speaking, this indicates an appropriate fit for the model.

model_fit <- pR2(guns_multinom)
paste("McFadden's Pseudo-R2:", round(model_fit[['McFadden']], 4))
## [1] "McFadden's Pseudo-R2: 0.3915"
