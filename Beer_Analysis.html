Data Processing & Cleaning

First thing first, we need to process and clean the data. The data is already fairly clean, so not much needs to be done except for a simple join and some name changes.

ABV Content

Another important aspect of beer is the ABV: the percentage of alcohol content. Below we see a distribution of the ABV values among our sample. Most ABV values are within the 0.05 mark, with a heavy rightward tail indicating more beers with higher alcohol content.

ggplot(beers) +
  geom_density(mapping = aes(abv), color = 'blue') + 
  labs(x = 'ABV', y = 'Density')

Just out of curiosity, we show the ABV distributions for three of our most common brews; and we can see that the American IPA has quite the kick.

styles <- c('American IPA',

Hypothetically, lets say I really like my alcohol with some bite. Which beer styles would satiate my indulgent vice? According to the plot below, one of the safest bets would be to go with imperial stouts.

Note: the plot only includes beer styles which have at least 5 selections, to account for some variability.

beers %>%
  group_by(style) %>%
  summarise(med_abv = median(abv), n = n()) %>%
  filter(n >= 5) %>%
  arrange(desc(med_abv)) %>%
  filter(row_number() <= 10) %>%
  ggplot(mapping = aes(x = reorder(style, med_abv), y = med_abv)) + 
    geom_point(size = 2.5, color = 'blue') + 
  labs(x = 'Beer Styles', y = 'Median ABV') + 
  ylim(.07, .10) + 
  coord_flip()

My Personal Favorites

Like I said previously, I enjoy craft beer, but only a select few: IPAs, Stouts, Porters, and Ciders. So I thought Id quickly take a peek at how these brews compare and contrast.

Since the beer styles vary, well have to create another column to quantify my selections.

favs <- beers %>%
  mutate(favorite = ifelse(str_detect(style, 'IPA'), 'IPA',
                    ifelse(str_detect(style, 'Stout'), 'Stout',
                    ifelse(str_detect(style, 'Porter'), 'Porter',
                    ifelse(str_detect(style, 'Cider'), 'Cider', 'Other'))))) %>%
  filter(favorite != 'Other')

That will make things easier. Now, the plot below tells me which breweries I should visit by counting the number of my favorite beers that are brewed by each brewery.

It looks like Brewery Vivant and Oskar Blues Brewery would be excellent choices.

favs %>%

What about the ABV value among the 4 libations, how do they compare? Ill run some quick analysis to see if there are any significant differences.

First, a box plot provides evidence that there are differences, and that the spread of the data look fairly similar among the different beer types.

ggplot(data = favs) +
  geom_boxplot(mapping = aes(favorite, abv)) +
  labs(x = 'Beers', y = 'ABV')

Next, we fit an ANOVA and see that the residuals appear to be normally distributed.

anova_style <- aov(abv ~ favorite, data = favs)
anova_res <- tibble(res = resid(anova_style))

par(mfrow = c(1,2))
ggplot(data = anova_res) +
  geom_density(mapping = aes(res), lwd = 1) +
  labs(x = 'Beers', y = 'Density', title = 'Density')

ggplot(data = anova_res) +
  geom_qq(mapping = aes(sample = res)) +
  labs(x = 'Theoritical', y = 'Sample', title = 'QQPlot')

Then lastly, we check the ANOVA output and see that there is indeed a significant difference in ABV among the beer styles (with a p-value of <0.001). And in fact, there is a significant difference of almost 1% among all the beers except Stouts vs.IPAs, and Porters vs.Ciders.
